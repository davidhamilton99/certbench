Question: At which OSI layer does a router primarily operate when making forwarding decisions?
A) Layer 2 — Data Link
B) Layer 3 — Network
C) Layer 4 — Transport
D) Layer 1 — Physical
Correct: B
Explanation: Routers operate at Layer 3 (Network) and use IP addresses to make packet-forwarding decisions between networks. Layer 2 devices like switches use MAC addresses, which makes them a tempting answer but they do not perform routing between networks. Remember: routers route packets at Layer 3, switches switch frames at Layer 2.

B) Segment
C) Frame
D) Bit
Correct: C

Question: During the encapsulation process, in what order is data processed as it travels down the OSI stack from the sending application?
A) Data → Segment → Packet → Frame → Bits
B) Bits → Frame → Packet → Segment → Data
C) Packet → Segment → Frame → Data → Bits
D) Data → Frame → Packet → Segment → Bits
Correct: A
Explanation: Encapsulation proceeds from upper to lower layers: Application data is segmented at Layer 4, then packaged into packets at Layer 3, framed at Layer 2, and finally converted to bits at Layer 1. Option B describes decapsulation (receiving side), not encapsulation. Think of encapsulation as wrapping a gift — each layer adds its own wrapper as data descends.

Question: Which OSI layer is responsible for decapsulation on the receiving host, stripping the Layer 2 frame header?
A) Layer 1 — Physical
B) Layer 3 — Network
C) Layer 4 — Transport
D) Layer 2 — Data Link
Correct: D
Explanation: The Data Link layer (Layer 2) processes the incoming frame, checks for errors using the Frame Check Sequence (FCS), strips the frame header and trailer, and passes the packet up to Layer 3. Layer 1 simply receives the raw bit stream and passes it to Layer 2; Layer 1 does not perform header stripping. Each layer in the receiving stack strips its own header as data moves upward.

Explanation: POP3 (Post Office Protocol version 3) uses TCP port 110 to retrieve email from a mail server, typically downloading and deleting messages from the server. TCP port 143 is IMAP and TCP port 25 is SMTP — all email protocols that are commonly confused. TCP port 995 is POP3S (POP3 over SSL/TLS), the secure version of POP3.

Domain: 1.0 Networking Concepts
Sub-objective: 1.7 — Basic corporate and datacenter network architecture
Question: Which protocol is used to prevent switching loops in a network with redundant switch connections?
A) OSPF
B) EIGRP
C) STP (Spanning Tree Protocol)
D) HSRP
Correct: C
Explanation: Spanning Tree Protocol (STP) and its successors (RSTP, MSTP) prevent Layer 2 switching loops in networks with redundant paths by blocking certain switch ports, ensuring only one active path exists at a time. OSPF and EIGRP are Layer 3 routing protocols that prevent routing loops but not switching loops. STP elects a root bridge and blocks redundant paths, activating them automatically if the primary path fails.

Domain: 1.0 Networking Concepts
Sub-objective: 1.1 — Explain concepts related to the Open Systems Interconnection (OSI) reference model
Question: SSL and TLS encryption protocols primarily operate at which OSI layer?
A) Layer 4 — Transport
B) Layer 7 — Application
C) Layer 3 — Network
D) Layer 6 — Presentation
Correct: D
Explanation: SSL/TLS operates at Layer 6 (Presentation) because it provides data encryption and decryption services — a presentation layer function — transparently to applications above it. Layer 4 (Transport) is a common distractor because TLS is often associated with TCP, and the acronym includes "Transport," but TLS provides presentation-layer encryption, not transport-layer functions. HTTPS, for example, is HTTP (Layer 7) encrypted by TLS (Layer 6) over TCP (Layer 4).

Question: Which of the following is a connection-oriented protocol that guarantees delivery through acknowledgments and retransmissions?
A) UDP
B) ICMP
C) TCP
D) IP
Correct: C

Question: A technician wants to determine which process on a Windows server is listening on TCP port 443. Which command would provide this information?
A) ipconfig /all
B) arp -a
C) netstat -ano
D) tracert 443
Correct: C
Explanation: The command netstat -ano on Windows shows all active connections and listening ports (-a), in numeric format (-n), with the owning Process ID (-o), allowing the technician to identify which PID is using port 443 and then look it up in Task Manager. Ipconfig only shows IP configuration and arp shows MAC-to-IP mappings — neither shows process-to-port associations. On Linux, the equivalent command is ss -tlnp or netstat -tlnp.

Question: North-south traffic in a traditional enterprise data center primarily refers to:
A) Traffic between servers on the same network segment
B) Traffic flowing between servers in different VLANs
C) Traffic flowing between external clients and internal servers across the perimeter
D) Traffic flowing between two data centers in different geographic locations
Correct: C

Domain: 1.0 Networking Concepts
Sub-objective: 1.8 — Summarize cloud concepts and connectivity options
Question: Which of the following BEST describes the "broad network access" characteristic of cloud computing?
A) Cloud resources can scale to meet any level of demand
B) Cloud services are accessible over the network via standard mechanisms from diverse client platforms
C) Cloud infrastructure is shared among multiple tenants
D) Cloud resources are provisioned without human interaction
Correct: B

Scalability to meet demand is elasticity, and infrastructure sharing among tenants is multitenancy. Broad network access ensures cloud services are platform-agnostic and location-independent.
