Question: An application uses MD5 to hash passwords stored in a database. A data breach exposes the hashed passwords, and attackers crack them using precomputed rainbow tables within hours. Which vulnerability is present?
A) SQL injection enabling database access
B) Cryptographic vulnerability — use of weak, broken hash algorithm
C) Misconfiguration — passwords not salted
D) Improper error handling
Correct: B
Explanation: MD5 is a cryptographically broken algorithm — it is fast (enabling rapid brute force) and has known collision vulnerabilities.
