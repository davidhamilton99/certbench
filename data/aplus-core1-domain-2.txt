Question: A network administrator needs to connect a 100Base-T device to a Gigabit Ethernet switch. The switch auto-negotiates the connection speed. What is the result?
A) The connection fails because speeds cannot be mixed
B) The switch port operates at 100 Mbps to match the slower device
C) The switch forces the device to upgrade to 1 Gbps
D) The device operates at half-duplex only
Correct: B
Explanation: Ethernet auto-negotiation (IEEE 802.3u) allows switch ports to detect the highest common speed and duplex setting supported by both ends and negotiate down to the lower capability. The device will not be forced to a speed it does not support, and the connection will succeed at 100 Mbps.

Sub-objective: 2.7 — Compare and contrast internet connection types, network types, and their features
Question: An organisation needs to connect two office campuses in different cities over a private, dedicated network path. Which WAN technology provides guaranteed bandwidth and is not shared with other customers?
A) Public internet VPN
B) Cable broadband
C) Dedicated leased line (e.g., T1, E1, or dark fibre)
D) 4G LTE cellular
Correct: C
Explanation: Dedicated leased lines (such as T1, E1, DS3, or dark fibre) provide a private, guaranteed-bandwidth circuit between two locations that is not shared with other customers, ensuring consistent latency and throughput. Public internet VPNs share internet infrastructure and are subject to variable congestion. Cable broadband is a shared medium within a neighbourhood segment. Cellular is shared wireless spectrum.

Domain: 2.0 Networking
Sub-objective: 2.5 — Install and configure basic wired/wireless SOHO networks
Question: A technician is configuring a SOHO router to segment a guest wireless network from the main corporate network. Which feature should be enabled to prevent guest users from accessing internal resources?
A) MAC address filtering on the guest SSID
B) Guest network VLAN isolation / guest network mode
C) WPS on the guest SSID
D) DNS filtering on the main SSID
Correct: B
Explanation: Most SOHO routers with a guest network feature place guest SSID traffic on a separate VLAN that is isolated from the main LAN, allowing guests to access the internet while being blocked from seeing internal hosts, printers, and NAS devices. MAC filtering only controls which devices can connect, not what they can access after connecting. WPS is a connection convenience feature and DNS filtering affects web content, not network segmentation.

Domain: 2.0 Networking
Sub-objective: 2.1 — Compare and contrast TCP and UDP ports, protocols, and their purposes
Question: A network administrator sees traffic destined for port 3389 being blocked by the perimeter firewall on inbound rules. A remote worker reports they cannot access their office desktop remotely. Which protocol requires port 3389 to be permitted?
A) SSH
B) VNC
C) RDP
D) Telnet
Correct: C
Explanation: RDP (Remote Desktop Protocol) uses TCP port 3389 for graphical remote desktop access to Windows systems. The firewall blocking inbound port 3389 prevents external RDP connections. The recommended approach is not to expose RDP directly to the internet (due to brute-force and exploit risk) but instead to use a VPN first and connect via RDP over the VPN tunnel. SSH uses port 22, VNC uses port 5900, and Telnet uses port 23.

Domain: 2.0 Networking
Sub-objective: 2.6 — Compare and contrast common network configuration concepts
Question: A workstation receives its IP configuration from DHCP with a lease time of 8 hours. After 4 hours (50% of the lease), what does the DHCP client attempt to do?
A) Release the IP address and request a new one
B) Renew the lease by contacting the DHCP server directly with a unicast request
C) Switch to an APIPA address as a backup
D) Broadcast a new DHCP Discover to find an alternate server
Correct: B
Explanation: At 50% of the lease time (T1), the DHCP client attempts to renew its lease by sending a unicast DHCPREQUEST directly to the original DHCP server. If the server responds with a DHCPACK, the lease is renewed. If renewal fails at T1, the client tries a broadcast renewal at T2 (87.5% of lease time). The client only falls back to APIPA if the lease fully expires without successful renewal and no DHCP server is found.

Domain: 2.0 Networking
Sub-objective: 2.8 — Use networking tools
Question: A technician is monitoring network traffic on a switch and needs to capture packets sent between a specific workstation and a server for analysis. The switch does not have a built-in packet capture feature. Which approach should the technician use?
A) Install a Wi-Fi analyser on the workstation
B) Use port mirroring (SPAN) on the switch to copy traffic to a monitoring port connected to a packet analyser
C) Connect a loopback plug to the server's NIC
D) Use a cable tester on the link between the workstation and switch
Correct: B
Explanation: Port mirroring (SPAN — Switched Port Analyser) configures the switch to copy all traffic from one or more source ports to a designated mirror (destination) port, where a device running a packet analyser (such as Wireshark) captures the traffic for analysis. A network hardware tap is an alternative that provides a similar function without relying on switch software. A loopback plug tests port functionality, and cable testers test physical cable integrity.

Domain: 2.0 Networking
Sub-objective: 2.3 — Compare and contrast protocols for wireless networking
Question: A warehouse deploys a wireless network for barcode scanners that frequently roam between access points throughout the facility. Which feature ensures seamless handoff between APs without dropping the connection?
A) SSID hiding
B) MAC address filtering
C) Fast BSS Transition (802.11r) / seamless roaming
D) WPS push-button pairing
Correct: C
Explanation: 802.11r (Fast BSS Transition) enables fast secure roaming by pre-establishing the security keys with the target AP before the client fully transitions, dramatically reducing the handoff time compared to full re-authentication. This is critical for real-time applications and constantly roaming devices like warehouse scanners where even brief connection interruptions cause data loss. SSID hiding and MAC filtering are security configurations that do not affect roaming behaviour.

Domain: 2.0 Networking
Sub-objective: 2.2 — Compare and contrast common networking hardware
Question: A network administrator is reviewing the benefits of using a managed switch over an unmanaged switch. Which capability is exclusive to a managed switch?
A) Connecting multiple devices via Ethernet ports
B) Forwarding frames based on MAC addresses
C) Configuration of VLANs, port security, and SNMP monitoring
D) Providing PoE power to connected devices
Correct: C
Explanation: Managed switches provide a full management interface (CLI, web GUI, or API) that allows administrators to configure VLANs, port security, QoS, SNMP monitoring, port mirroring, STP settings, and link aggregation. Unmanaged switches forward frames using MAC addresses and support PoE (if PoE-capable) but offer no configuration interface. The ability to segment, monitor, and control network behaviour is the primary value of managed switches in enterprise environments.
