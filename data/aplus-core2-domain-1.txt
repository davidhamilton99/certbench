Explanation: Dark mode in Windows 10 is enabled in Settings > Personalisation > Colours, where "Choose your colour" can be set to "Dark" (dark mode for apps and system UI) or "Custom." High Contrast in Ease of Access is a different accessibility feature for users with visual impairments, not the same as dark mode. "Dark mode = Settings > Personalisation > Colours > Dark."

Question: A Windows 10 workstation configured with a static IP cannot browse the internet but can access local network resources. Which setting is most likely misconfigured?

B) Default gateway

Question: A technician performs a network-based Windows installation using a WDS server. What protocol does the client use to obtain the boot image from the server before the OS loads?
A) HTTP
B) FTP
C) TFTP over PXE
D) SMB
Correct: C
Explanation: PXE boot uses DHCP to locate the boot server and then uses TFTP (Trivial File Transfer Protocol) to download the network boot program (NBP) and Windows PE boot image from the WDS server. SMB is used after booting to access the WDS installation share, not for the initial boot image transfer. "PXE boot = DHCP to find server + TFTP to download boot image."

Question: Which statement correctly describes the difference between a 32-bit and 64-bit operating system installation?
A) A 64-bit OS can run 32-bit applications but a 32-bit OS cannot run 64-bit applications natively
B) A 32-bit OS can run 64-bit applications in compatibility mode
C) Both OS types support the same maximum RAM
D) 64-bit applications run on 32-bit OS using WOW64
Correct: A
Explanation: A 64-bit OS can run both 32-bit (via WOW64) and 64-bit applications, but a 32-bit OS can only run 32-bit applications — 64-bit apps cannot run on a 32-bit OS. WOW64 is a subsystem of 64-bit Windows that enables 32-bit apps to run, not the reverse. "64-bit OS = runs both; 32-bit OS = 32-bit apps only."
