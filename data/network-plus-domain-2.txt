Question: OSPF is classified as which type of routing protocol?
A) Distance-vector
B) Path-vector
C) Link-state
D) Hybrid
Correct: C
Explanation: OSPF (Open Shortest Path First) is a link-state protocol in which each router builds a complete map of the network topology using Link State Advertisements (LSAs) and runs the Dijkstra algorithm to calculate the shortest path. OSPF converges faster than distance-vector protocols and is free from routing loops by design because each router has a consistent topology database.
