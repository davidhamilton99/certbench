A) Full
B) Differential

The recovery process takes longer as more incremental sets must be applied in sequence.

Question: A technician wants to create a script that is compatible across multiple operating systems including Windows, macOS, and Linux. Which scripting language is MOST appropriate?
A) VBScript (.vbs)
B) Batch (.bat)
C) Python (.py)
D) PowerShell (.ps1) with Restricted policy
Correct: C
Explanation: Python is a cross-platform scripting language that runs natively on Windows, macOS, and Linux with minimal modification between environments. Its extensive standard library supports file operations, network communication, and system management. VBScript and .bat are Windows-only, and while PowerShell is now cross-platform, it is less universally available by default on non-Windows systems.

Question: A technician needs to remotely control a Windows workstation's graphical desktop to troubleshoot an application. Which protocol is natively built into Windows for this purpose?
A) SSH
B) VNC
C) RDP (Remote Desktop Protocol)
D) MSRA
Correct: C
Explanation: Remote Desktop Protocol (RDP) is Microsoft's proprietary protocol that provides a full graphical desktop session to a remote Windows machine. It operates on TCP port 3389 by default and supports audio, clipboard sharing, and drive redirection. RDP is built into all modern Windows editions and is the standard enterprise tool for remote Windows administration.

Question: A remote worker needs secure command-line access to a Linux server to manage files and run commands. Which remote access technology is MOST appropriate?
A) RDP
B) VNC
C) SSH (Secure Shell)
D) MSRA
Correct: C
Explanation: SSH provides encrypted, authenticated command-line access to remote systems over TCP port 22. It is the standard tool for securely managing Unix/Linux servers remotely. SSH supports file transfers via SCP and SFTP, tunnelling, and key-based authentication, making it secure and versatile.

It is built into Windows and requires the user's explicit consent for each session.

Question: Which remote access protocol transmits data in plain text and should NOT be used for remote administration of servers because of significant security risks?
A) SSH
B) RDP
C) Telnet
D) SFTP
Correct: C

It has been replaced by SSH, which provides encrypted communication. Any environment using Telnet for remote server management is exposing credentials and sensitive data to potential interception.

Sub-objective: 4.9 — Given a scenario, use remote access technologies
Question: An RDP session to a remote workstation is connecting successfully, but a technician is concerned about security. Which of the following is a BEST practice for securing RDP access?
A) Leave RDP on its default port 3389 with no additional configuration
B) Disable encryption and use plain-text mode for faster connections
C) Change the default RDP port, use Network Level Authentication (NLA), and restrict access via firewall
D) Allow RDP access from all IP addresses to ensure technicians can connect from any location
Correct: C
Explanation: Securing RDP involves multiple layers: changing the default port reduces automated scanning attacks, Network Level Authentication (NLA) requires credentials before a full session is established, and firewall rules restrict which IP addresses can initiate RDP connections. Together, these measures significantly reduce the attack surface of an exposed RDP service. RDP exposed directly to the internet without these controls is a leading vector for ransomware attacks.

Sub-objective: 4.3 — Given a scenario, implement workstation backup and recovery methods
Question: A technician is reviewing a company's backup strategy and notices that the most recent backup is always overwritten by the next backup cycle, with no historical copies retained. What critical limitation does this strategy have?
A) It uses too much storage space
B) It provides no ability to recover from data corruption that existed at the time of the last backup
C) It violates the 3-2-1 rule by using too many backup locations
D) It only works with full backup types
Correct: B
Explanation: A strategy that overwrites the only backup every cycle means that if data becomes corrupted before the corruption is noticed, the backup itself will contain the corrupted data after the next run. Retaining multiple generations of backups (such as through grandfather-father-son rotation) allows recovery to a point before the corruption occurred. A single overwriting backup provides minimal protection against gradual data corruption.

Question: A technician accidentally drops a CPU onto a carpeted floor. The room is very dry and the technician feels a slight tingle when retrieving it. What is the MOST likely concern?
A) The CPU pins have been damaged beyond repair
B) The CPU has been exposed to ESD that may have caused internal damage
C) The carpet fibres have contaminated the CPU's thermal interface
D) The capacitors in the CPU have discharged on impact
Correct: B
Explanation: Carpet combined with low humidity is an ideal environment for generating static electricity. A perceptible tingle indicates a static discharge event, which may have subjected the CPU to thousands of volts even if the technician only felt a mild sensation. ESD damage to a CPU may not cause immediate failure but can lead to intermittent issues or premature failure later.

Question: A UPS alarm indicates that its battery capacity has dropped to 20% during a power outage, and the runtime remaining is less than 2 minutes. What should the technician do immediately?
A) Continue working and trust that the UPS will sustain the system
B) Initiate a graceful shutdown of all connected systems before the UPS is exhausted
C) Disconnect the UPS to preserve battery life for a future outage
D) Replace the UPS battery while the system is still running
Correct: B
Explanation: When a UPS signals critically low battery, the priority is to perform a controlled shutdown of all connected systems before power is lost entirely. An unexpected power cutoff when the UPS dies can cause data corruption, filesystem damage, and hardware failure. Most UPS units can be configured to trigger an automatic graceful shutdown when battery reaches a defined threshold.

Question: A technician working from home needs to access a remote workstation in the office, but the company's firewall blocks inbound RDP connections from the internet. A VPN is not available. Which alternative approach would allow the technician to connect?
A) Use Telnet to bypass the firewall on port 23
B) Use a reverse connection tool or cloud-based relay remote desktop solution
C) Ask the office user to email the desktop screenshot regularly
D) Enable RDP on port 80 to pass through as web traffic
Correct: B
Explanation: Cloud-based relay remote desktop tools (such as TeamViewer or AnyDesk) establish outbound connections from the target machine to a cloud relay server, allowing the remote technician to connect through the relay without requiring inbound firewall rules. This model is specifically designed to work through NAT and firewalls that block inbound connections. Using port 80 for RDP would be a security policy violation and may not be reliable.
